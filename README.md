
<h2>LetsChat</h2>

<h1>Chatting with Friends</h1>

<p>This is a website development project for chatting with friends, built using the Django framework and SQLite3 database.</p>

<h2>Installation</h2>

<p>To run the program, you need to have Python and Django installed on your computer. Here are the steps to get started:</p>

<ol>
  <li>Create a virtual environment using virtualenv or venv.</li>
  <pre><code>virtualenv venv</code></pre>
  <li>Activate the virtual environment.</li>
  <pre><code>.\venv\Scripts\activate</code></pre>
  <li>Install the required packages using the req.txt file.</li>
  <pre><code>pip install -r req.txt</code></pre>
  <li>Install the Redis channel layer in your pc.</li>
  <li>Run the command python manage.py runserver.</li>
  <pre><code>python manage.py runserver</code></pre>
</ol>

<p>That's it! You should now be able to access the website and start chatting with your friends.</p>
